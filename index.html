IPCONFIG

Affiche l'adresse IP, le masque de sous-réseau et la passerelle par défaut de chaque carte liée à TCP/IP.
OPTIONUTILISATIONEXEMPLE
/?Affiche un message d'aideipconfig /?
/allAffiche toutes les informations de configurationipconfig /all
/releaseLibère l'adresse IPv4 pour la carte spécifiéeipconfig /release
/release6Libère l'adresse IPv6 pour la carte spécifiéeipconfig /release6
/renewRenouvelle l'adresse IPv4 pour la carte spécifiéeipconfig /renew
/renew6Renouvelle l'adresse IPv6 pour la carte spécifiéeipconfig /renew6
/displaydnsAffiche le contenu du cache de résolution DNSipconfig /displaydns
/flushdnsVide le cache de résolution DNSipconfig /flushdns
